# Excel Form Filler - Приложение для заполнения Excel форм

## Описание

Приложение Windows Forms для удобного заполнения данных и их сохранения в Excel файлы с использованием библиотеки EPPlus.

## Функциональность

- **Создание Excel файла**: Создает новый Excel файл с заголовками
- **Заполнение данных**: Записывает данные из формы в Excel файл
- **Валидация**: Проверка корректности введенных данных (обязательные поля, Email)
- **Удобный интерфейс**: Интуитивно понятная форма с группировкой полей

## Поля формы

### Личная информация
- Имя (обязательно)
- Фамилия (обязательно)
- Дата рождения

### Контактная информация
- Email (с валидацией)
- Телефон
- Адрес

### Дополнительная информация
- Должность
- Отдел
- Заметки

## Требования

- .NET Framework 4.7.2
- Windows 10/11
- EPPlus 7.1.2

## Инструкция по использованию

1. **Сборка проекта**:
   - Откройте решение `ExcelWF.slnx` в Visual Studio
   - Выберите конфигурацию Debug или Release
   - Нажмите Build → Build Solution (Ctrl+Shift+B)

2. **Запуск приложения**:
   - Запустите скомпилированный файл `ExcelWF.exe`
   - Или нажмите F5 в Visual Studio для отладки

3. **Использование**:
   - Нажмите кнопку "Создать Excel файл" для создания нового файла
   - Заполните необходимые поля формы
   - Нажмите "Заполнить данные" для сохранения в Excel
   - Повторяйте для добавления новых записей
